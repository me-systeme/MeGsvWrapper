# MeGsvWrapper
GSV DLL C# Wrapper

A simple C# wrapper for **MEGSV.dll**, used to communicate with GSV measurement devices.

## Project Structure

- **MeGsvWrapper**  
  C# class library that wraps the native `MEGSV.dll` functions.

- **TestConsole**  
  Small console application used to test the wrapper and demonstrate how to use it.
